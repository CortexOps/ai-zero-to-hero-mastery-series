# Beginner Mindset

You do NOT need:

* Coding experience
* Advanced technology knowledge
* Expensive hardware

You DO need:

* Curiosity
* Practice
* Ethical awareness
* Willingness to learn

---

# Safety Reminder

Always:

* Verify AI responses
* Protect private information
* Use critical thinking
* Practice responsibly
