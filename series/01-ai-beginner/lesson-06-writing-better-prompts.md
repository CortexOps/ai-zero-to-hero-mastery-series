# Why Prompting Matters

AI output quality often depends heavily on input quality.

## Simple truth:

> Better prompts = Better results

---

# Weak Prompt Example

```text id="u6wp9d"
Write an email.
```

### Problems:

* No context
* No audience
* No tone
* No goal

---

# Strong Prompt Example

```text id="r2yn8k"
Write a professional email to my supervisor requesting one day off next Friday for a medical appointment.
```

### Improvements:

* Clear audience
* Clear purpose
* Clear tone
* Clear context

---

# Core Prompt Formula

## Good prompts often include:

* Task
* Context
* Audience
* Tone
* Format
* Constraints

---

# Prompt Template

```text id="2r0fgx"
Act as [role].
Help me [task].
The audience is [audience].
Use [tone].
Format as [format].
Include [specific requirements].
```

---

# Example

```text id="9hv2qa"
Act as a career coach.
Help me improve my resume summary.
Audience is hiring managers.
Use confident professional tone.
Format as bullet points.
```

---

# Beginner Prompt Upgrades

## Instead of:

```text id="1mw5gf"
Help with resume
```

## Use:

```text id="3b7czt"
Improve my IT systems administrator resume summary for leadership roles in education technology.
```

---

# Important Prompt Skills

## Be specific

More detail usually improves quality.

## Be clear

Avoid vague wording.

## Set expectations

Define output style.

## Iterate

Refine prompts as needed.

---

## Learning:

# Prompting Mistakes

* Too vague
* Too broad
* Missing context
* No audience
* No desired format
* Unrealistic expectations
