# Common Mistakes

* Poor change control
* Weak backups
* No redundancy
* Lack of documentation
* Reactive maintenance
* Misconfigured permissions
* Weak segmentation
* Capacity neglect

Strong administrators become trusted because they:

* Prevent outages
* Scale environments
* Reduce downtime
* Improve efficiency
* Build resilience
* Strengthen security
